## Project setup <br>

```
# Clone the repository
$ git clone https://github.com/mateusdayrell/phone-book.git

# Access the project folder
$cd phone-book

# Install the dependencies
$ npm install

# Copy .env.example to .env
$ cp .env.example .env

# Serve aplication
$ npm start
```
